Website for a restaurant

I created this project to learn more about Python development.

How it works: The website is a Web App connected to a Heroku server that hosts the SQLite database, which defines the pizza flavors that will be listed. The flavors, ingredients and prices can be changed at the admin panel on the Heroku server.
There is also a Desktop App, built with Python's Kivy framework, that exhibits the same information. In a near future I intend to add more content and build a Mobile App.

Technologies: HTML, CSS, PYTHON, DJANGO, SQLite, API, JSON, GIT, HEROKU, KIVY
